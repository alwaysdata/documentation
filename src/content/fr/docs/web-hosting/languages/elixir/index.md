Le fonctionnement d'[Elixir](https://elixir-lang.org/) chez alwaysdata est très standard. Si vous avez l'habitude d'utiliser Elixir sous un système Unix (par exemple, Linux), vous savez déjà presque tout ce qu'il faut.

* les [versions proposées](/fr/docs/hebergement-web/langages/elixir/configuration/#versions-supportées) vont de la 1.13 à la 1.18 ;
* la [configuration d'environnement](/fr/docs/hebergement-web/langages/elixir/configuration/#environnement) se fait simplement.

Notre marketplace propose l'installation du framework [Phoenix](https://www.phoenixframework.org/).

* [Ressource API](https://api.alwaysdata.com/v1/environment/elixir/doc/)
