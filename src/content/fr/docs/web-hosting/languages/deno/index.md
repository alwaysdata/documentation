Le fonctionnement de [Deno](https://deno.land/) chez alwaysdata est très standard. Si vous avez l'habitude d'utiliser Deno sous un système Unix (par exemple, Linux), vous savez déjà presque tout ce qu'il faut.

- Plusieurs [versions](/fr/docs/hebergement-web/langages/deno/configuration/#versions-supportées) sont disponibles ;
- la [configuration d'environnement](/fr/docs/hebergement-web/langages/deno/configuration/#environnement) se fait simplement.
