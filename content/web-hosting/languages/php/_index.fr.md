Le fonctionnement de [PHP](https://www.php.net) chez alwaysdata est très standard. Si vous avez l'habitude d'utiliser PHP sous un système Unix (par exemple, Linux), vous savez déjà presque tout ce qu'il faut.

* les [versions proposées](/web-hosting/languages/php/configuration#versions-supportées) vont de la 4.4 à la 8.5 ;
* il est tout à fait possible de [personnaliser le fichier php.ini](/web-hosting/languages/php/configuration#paramètres-phpini) ;
* PHP tourne en *FastCGI*, derrière [Apache](/web-hosting/sites/configure-apache) : il est donc possible de créer des fichiers `.htaccess` ;
* il est possible d'[installer des extensions](/web-hosting/languages/php/extensions) (depuis PECL ou autre) ;
* il est possible d'[installer des paquets](/web-hosting/languages/php/packages) (*Composer*, *PEAR*) ;
* il est possible d'[installer des bibliothèques](/web-hosting/languages/php/libraries).

Notre marketplace propose l'installation des frameworks [CakePHP](https://cakephp.org/), [Laminas](https://getlaminas.org/), [Laravel](https://laravel.com/), [Symfony](https://symfony.com/) et [TNH](https://github.com/tnhfw/tnh-fw).
