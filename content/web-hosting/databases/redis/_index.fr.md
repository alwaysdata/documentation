## Connexion

|             |                              |
|-------------|------------------------------|
| **Serveur** | localhost ou 127.0.0.1       |
| **Port**    | 6379 (Port Redis par défaut) |

Redis ne tourne qu'en local avec des droits isolés (utilisateur distinct).

{{% notice info %}}
Cette configuration est disponible sur le [Cloud Privé](/admin-billing/billing/private-cloud-prices). Sur le [Cloud Public](/admin-billing/billing/public-cloud-prices) vous pouvez le faire tourner en suivant ce [guide](/development/guides/redis).
