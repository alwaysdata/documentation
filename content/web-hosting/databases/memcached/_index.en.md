## Connection

|||
|--- |--- |
|Server|localhost or 127.0.0.1|
|Port|11211 (Memcached port by default)|


Memcached only runs in local mode with isolated permissions (separate user).

{{% notice info %}}
Memcached is only available on [Private Cloud](/admin-billing/billing/private-cloud-prices) environments. On the [Public Cloud](/admin-billing/billing/public-cloud-prices) you can run it by following this [guide](/development/guides/memcached).
